# Change summary schema

For each edit:

| Field | Values |
|-------|--------|
| Location | Section + employer or heading |
| Type | `rephrase` \| `reorder` \| `emphasize` \| `trim` |
| Before | Short quote from base |
| After | Short quote from tailored |
| JD link | Requirement or keyword driving change |
| Suggested emphasis | Optional tag when reframing without new facts |

Omit changes that only fix typos unless JD-relevant.
