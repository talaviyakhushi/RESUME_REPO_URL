# JD extraction taxonomy

Classify each JD line as:

| Bucket | Use in fit report |
|--------|-------------------|
| Must-have | Strong matches or Gaps |
| Preferred | Gaps if missing |
| Responsibilities | Strong matches when evidenced |
| Tools/stack | Skills alignment |
| Nice-to-have | Gaps only if relevant |

Map rows to **Strong matches**, **Gaps**, or **Do not add**.
